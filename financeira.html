Matemática Financeira

Enquanto você lê, esculte essa música:

    A matemática financeira é a área da matemática que estuda a equivalência de capitais no tempo, ou seja, como se comporta o valor do dinheiro no decorrer do tempo. Sendo um área aplicada da Matemática, estuda diversas operações ligadas ao dia a dia das pessoas. Por esse motivo, conhecer suas aplicações é fundamental.

Juros Simples

    O juro simples é uma taxa previamente definida e que incide somente sobre o valor inicial.

    Veja a baixo as duas fórmulas do juros simples:

J = C * i * n

J=juros
C=capital inicial
i=taxa de juros
n=prazo

M = C (1 + i * n)

O Montante, dedica-se a estudar
e controlar a variação sofrida
por uma quantia investida ao longo
do tempo.


Exercícios

1.(Vunesp) Num balancete de uma empresa consta que certo capital foi aplicado a uma taxa de 30% ao ano durante 8 meses, rendendo juros simples no valor de R$ 192,00. Quanto foi o capital aplicado?

i = 30% ao ano;
n = 8 meses;
J = 192,00
i = 2,5% a.m
J = C · i · n
192 = C · 0,025 · 8
192 = 0,2 C
C = 192 / 0,2
C = 960

2.Um capital foi investido a juros simples com uma taxa de 12% a.a. Durante quanto tempo esse capital deve ficar investido para que o montante seja o quádruplo do capital?

M = C + J
4C = C + J
4C – C = J
3C = J
i = 12% a.a
J = 3C
J=C*i*n
3C= C * 0,12 * n
3C/C = 0,12 * n
3 = 0,12n
3/0,12 = n
n = 25 anos


Juros Composto

     Os juros compostos são aqueles nos quais os juros do mês são incorporados ao capital. Com uma taxa assim, o valor cresce muito mais rápido do que com juros simples. No caso de uma dívida, é perigoso. No caso de um investimento, é excelente.

M = C(1 + i)^n

M=montante
C=capital
i=taxa de juros compostos
n=números de períodos

A partir dessa fórmula, para
encontrar o i chegamos a
uma potência, e para o n
usamos normalmente o log.

     Obs: o n deve estar de acordo com a unidade de medida do i, se em semestre, devemos converter os dois em semestre, se em dias, os dois devem estar em dias, e assim por diante.


Exercícios

1.Qual é a taxa de juros aplicada ao ano para que um capital de R$ 8000 gere juros de R$ 3520, em dois anos, a juros compostos?

J = 3520
C = 8000
t = 2
M = 8000 + 3520 = 11.520
M = C(1 + i)n
11.520 = 8000 (1 + i)2
11.520 : 8000 = (1 + i)²
1,44 = (1 + i)²
√1,44 = 1 + i
1,2 = 1 + i
1,2 – 1 = i
i = 0,2

2.Um capital de R$ 1500 foi aplicado a juros compostos com taxa percentual de 2% a.a. Qual será o montante daqui a 2 anos?

C = 1500
i = 2% a.a.
t = 2 anos
M = C(1 + i)n
M = 1500 (1 + 0,02)²
M = 1500 (1,02)²
M = 1500 · 1,0404
M = 1560,60
